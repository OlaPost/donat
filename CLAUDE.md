# Инструкции для Claude

## Кто я

Владелец проекта, не разработчик. Идеи объясняю словами.
Не всегда знаю технические термины — объясняй понятно.

## Как со мной работать

- **Коротко и по делу** — без длинных вступлений
- **Объясняй зачем**, не только что делаешь
- **Если есть развилка** — предложи вариант сам и объясни почему
- **Перед большими изменениями** — спроси подтверждение
- **Язык интерфейса** — русский везде

## Технический стек

- React 18 + TypeScript + Vite
- Telegram WebApp SDK (CDN в index.html)
- CSS-переменные для темы (тёмная по умолчанию, iOS-стиль)
- Платёжка — пока не подключена

## Структура проекта

```
src/
  App.tsx              — главный компонент, управление экранами
  screens/
    AmountScreen.tsx   — выбор суммы
    MessageScreen.tsx  — сообщение автору
    PayScreen.tsx      — подтверждение и оплата
    ThankScreen.tsx    — экран благодарности
  index.css            — все стили (iOS dark theme)
```

## Правила кода

- Не создавай новые файлы без необходимости — редактируй существующие
- Не добавляй фичи сверх того, что просят
- Стили — в index.css, не inline
